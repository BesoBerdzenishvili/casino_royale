# Casino Royale

This project includes a game that can be launched directly from the UI, with dynamic language and style changes based on location selection.

## Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

In terminal run: `git clone https://github.com/BesoBerdzenishvili/casino_royale.git`

Enter the repository:
`cd your-repo-name`

### 2. Install dependencies

run `npm i`

### 3. Run the development server

run
`npm run dev`

## How to Play

Once the app is running, open it in your browser.

Click the OPEN THE GAME button to start playing.

## How to change location

You can select a new location from a dropdown menu. This will change the UI language and also update the styles dynamically.
